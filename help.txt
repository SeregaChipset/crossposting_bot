бот для кросс постинга

[старт меню] -> [*создание поста] -> [Напишите название] -> [Напишите описание] -> [Напишите цену] -> [Размещение]
       -> [*номер заказа] -> [Напишите номер заказа] -> [Выводит информацию о заказе]
       -> [*настройки] -> [Пример поста] -> [Выводим образец поста]
                      -> [Написать подвал] -> [Введите текст подвала]
                      -> [Токены] -> [Введите токен вк] -> [Введите токен телеграм]
       [Отмена] -> [Уходит в меню]

v1 - основной функционал. Создания поста, добавление подвала, размещение в вк и телеге. Просмотр заказов. Настройки.
v1.1 - рефакторинг
v1.2 - асинхронность запросов
v1.3 - добавлена возможность размещения товара. ссылка на вк в посте тг
